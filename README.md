## 项目管理

### Git 分支规范
- 主分支(发布分支): `master`
- 开发分支主干: `dev`
- 功能分支: `feature/*`
- 热修复分支: `hotfix/*`

### 项目目录规范

采用 Monorepo 策略进行管理：
- 前端项目目录: `frontend`
- 后端项目目录: `backend`
- 项目文档目录: `documents`
- 测试用例目录: `test`
